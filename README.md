# kitchen69

REST API สำหรับจัดการเมนูอาหาร พัฒนาด้วย Go

## Features
- GET /menu
- GET /menu/{id}
- POST /menu
- DELETE /menu/{id}
- Filter ด้วย `?type=`
- CORS Support
